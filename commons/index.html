Beangle Commons ​

Beangle Commons 是一个基于 Scala 的企业级工具库，提供 24 个核心模块，覆盖字符串处理、集合操作、文件IO、网络通信、安全加密等企业级开发常用功能。

Modules ​

  • API 调用说明 完整的API使用指南，包含模块总览和所有模块的详细说明
