# Beangle Notify

Beangle Notify 是一个基于 Scala 的通知框架，提供了邮件、短信等多种通知方式，支持异步发送、消息队列等功能，适用于企业级应用的通知需求。

## 核心能力

Beangle Notify 提供以下核心能力：

- **邮件通知**：支持发送文本和 HTML 邮件，支持附件
- **短信通知**：支持多种短信服务商，提供验证码服务
- **异步发送**：支持消息队列和异步处理
- **消息管理**：提供消息的创建、发送和跟踪
- **通知服务**：统一的通知服务接口，支持多种通知方式
- **可扩展性**：支持自定义通知方式和服务商
