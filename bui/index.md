# Beangle BUI

Beangle BUI (Beangle User Interface) 是一个基于 Bootstrap 的前端 UI 框架，提供了丰富的标签和组件，用于快速构建现代化的 Web 应用界面。

## 核心能力

Beangle BUI 提供以下核心能力：

- **标签库**：基于 Freemarker 的 UI 标签，提供表单、网格、导航等组件
- **主题支持**：内置 Bootstrap 主题，支持自定义主题
- **表单组件**：文本框、下拉框、日期选择器等丰富的表单元素
- **布局组件**：网格系统、卡片、导航栏等布局元素
- **工具类**：图标、消息、工具栏等实用组件
- **JavaScript 支持**：提供客户端交互和异步操作
